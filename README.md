- React 18 with TypeScript

- Shadcn/ui component library
- React Router for navigation
- Lucide React for icons

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
